        #최종발표에서 대본 앞부분을 가져왔어 파악하기에는 쉬울 거야
        In 2D interfaces, shortcuts like Ctrl+C and Ctrl+V are universally understood across all platforms and devices. 
        But in 3D interfaces, no such shared vocabulary exists yet.
        Every platform defines its own gestures, leading to fragmented experiences.
        On top of that, the closest existing system from TVCG 2025 only handles 2D text editing. 
        There's simply no system designed for object manipulation using 3D shortcut-based gestures. That's the gap we're addressing. 
        So what do we need? A 3D shortcut system that's familiar, intuitive, and universal: building on 2D knowledge and applicable to any shortcut rather than arbitrary mappings. 
        Our solution maps Ctrl+C, F, Z, X directly to ASL finger-spelling gestures for those letters. Simple, learnable, and grounded in an existing system. 
        Here's how the system works end-to-end. A camera captures both hands in real time. The system recognizes the gesture, whether Copy, Find, Undo, or Delete, distinguishing left and right hands for command processing. Once recognized, users select and place blocks in 3D space with a pinch interaction. The arrangement updates in real time, with visual feedback indicating match status throughout.
