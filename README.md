# Young Bin Kim — Personal Website

Personal academic website of Young Bin Kim (M.S. student, KAIST UVR Lab).

## 🔗 Live site

**https://youngbin0212.github.io/**

This is a GitHub Pages user site: once changes are pushed to the `main` branch
of `youngbin0212/youngbin0212.github.io`, they are automatically published at
the URL above (usually within 1–2 minutes).

## Structure

- `index.html` — the entire site (HTML + CSS + JS in one file)
- `images/` — profile photo and other image assets

## Local preview

Just open `index.html` in a browser (double-click it). An internet connection is
needed for the web fonts (Inter / Fraunces / Pretendard) to load.
